Kompleksiluvut

Kompleksiluvun määritelmä

Kompleksilukujen joukko $\mathbb{C}$ eli kompleksitaso ovat kunta, jonka alkioille $$z = x + yi = (x, y) \in \mathbb{R}^2$$ on määritelty
  • yhteenlasku $ (a, b) + (c, d) = (a + c, b + d)$
  • kertolasku $(a, b)(c, d) = (ac - bd, bc + ad)$.
Kompleksilukua $z \in \mathbb{C}$ merkitään tavallisesti $z = x + yi$, jolloin
  • $(a + bi) + (c + di) = a + c + (b + d )i$
  • $(a + bi) (c + di) = (ac - bd) + (bc + ad)i $.

Muita määritelmiä

Vaihtoehtoisia merkintöjä kompleksiluvulle $z$: $ x + y, \quad ix + iy, \quad x + y\textrm{i},\quad x + yj$
Kompleksiluvun reaaliosa $\operatorname{Re}(z) = \Re(z) = x$
Imaginääriosa $\operatorname{Im}(z) = \Im(z) = y$
Iteisarvo eli moduli (myös magnitudi) $r = |z| = |x+ yi| = \sqrt{x^2 + y^2}$
Vaihekulma eli argumentti $\phi = \operatorname{arg}(z); \quad \tan \phi = \frac{y}{x}, a \not = 0$
Liittoluku eli kompleksikonjugaatti $z^\ast = \overline{z} = \overline{x + yi} = x - yi$

Kompleksiluvun napaesitys

$$z = x + yi = |z| e^{i\phi} = |z| (\cos \phi + i \sin \phi) $$ Tunnetaan muodossa $ e^{i\phi} = \cos \phi + i \sin \phi$ myös Eulerin kaavana tai Eulerin lauseena (kompleksiluvuille).

Ominaisuuksia ja laskutoimituksia

Ykkösen neliöjuuri $i^2 = -1; \quad i = \sqrt{-1}$
Jakolasku $\frac{a + bi}{c + di} = \frac{ac + bd}{c^2 + d^2} + \frac{bc - ad}{c^2 + d^2} i$
Itseisarvon neliö $|z|^2 = x^2 + y^2 = (x + yi)(x - yi) = z\overline{z}$
Liittoluvun liittoluku $(z^\ast)^\ast = \overline{\overline{z}} = z$
Kokonaislukupotenssi (De Moivren kaava) $(z^n) = (r(\cos \phi + i \sin \phi))^n = r^n(\cos n\phi + i \sin n\phi)$
